Reptiles - rails app
====================

How to run the application?

Clone the repo:

    $ git clone https://github.com/pro-vagrant/reptiles-rails-app.git
    $ cd reptiles-rails-app

Boot VM:

    $ vagrant up
    $ vagrant ssh
    $ cd /vagrant

Install bundles and start the server:

    $ bundle install
    $ rails server -d

Finally start your web browser and visit:

    http://127.0.0.1:3000
